# Evidence

`campaign-summary.json` is the public index. `runs/` contains the exact safe subset of the current live run. `terminal-bench/` and `deep-swe/` contain selected harness evidence, curated private-source commitments, and derived summaries.

Files omitted from the private source are named in the repository security and limitation documents. A curated receipt exposes the raw file's SHA-256 and selected non-sensitive fields, but it remains an operator attestation unless the reader also holds the private source. No omitted file is silently represented as published.
